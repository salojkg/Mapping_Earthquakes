# Mapping_Earthquakes
## Overview

The Objective of this project is to view the  Earthquake data in retaion to the Tectonic Plates location on earth. In addition, this allow to see all earthquakes with magnitude > 4.5 on the map.

## Environment
- JavaScript
- Leaflet
- Mapbox
- CSS
- HTML
